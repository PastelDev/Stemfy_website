# Κατασκευαστής Γέφυρας

## Τι είναι;
Ένα παιχνίδι-προσομοίωση όπου σχεδιάζεις γέφυρες και τις δοκιμάζεις με πραγματική φυσική.

## Πώς παίζεται;

### Φάση Κατασκευής
- Η φυσική είναι **απενεργοποιημένη** — τα μπλοκ μένουν όπου τα βάζεις
- Τοποθέτησε 2D μπλοκ για να χτίσεις τη γέφυρά σου
- Συνδύασε, στοίβαξε, δημιούργησε

### Φάση Δοκιμής
- Ενεργοποίησε τη φυσική και δες τι γίνεται
- Σφαίρες διαφορετικών μαζών κυλούν πάνω στη γέφυρα
- Η βαρύτητα, η τριβή και οι συγκρούσεις είναι αληθινές

## Η Πρόκληση
Μπορείς να φτιάξεις μια γέφυρα που:
- Αντέχει τις σφαίρες χωρίς να καταρρεύσει;
- Επιτρέπει στις σφαίρες να περάσουν με επιτυχία;
- Δεν είναι τόσο απότομη που οι σφαίρες γλιστρούν πίσω;

## Αποθήκευση & Φόρτωση
- Αποθήκευσε τη γέφυρά σου ως αρχείο
- Φόρτωσε παλιές κατασκευές για να τις βελτιώσεις
- Μοιράσου το αρχείο με φίλους

## Διαγωνισμοί (Σύντομα!)
Θα οργανώνουμε προκλήσεις — φτιάξε την πιο τρελή γέφυρα που αντέχει! Ανέβασε screenshot με το αποτέλεσμα στα σχόλια.

## Η Φυσική
Η προσομοίωση χρησιμοποιεί:
- **Βαρύτητα**: τα αντικείμενα πέφτουν
- **Τριβή**: τα αντικείμενα δεν γλιστρούν αν η κλίση είναι μικρή
- **Συγκρούσεις**: τα αντικείμενα αλληλεπιδρούν ρεαλιστικά
